# 机器人项目技术栈
[你知道一个机器人项目都涉及哪些技术，怎么组织起来的吗？ - 陈不陈](https://zhuanlan.zhihu.com/p/634848960)

机械设计：
- 三维设计：SolidWorks，UG，CATIA等
- 动力学、运动学：
	- 技能：动力学分析（拉格朗日动力学等）
	- 软件：Adams，CoppeliaSim等
- 系统仿真：CoppeliaSim，Webots等
- 有限元仿真：Ansys，Abuqus等
硬件电路设计：
- 技能：模电数电；EMC/EMI分析
- 软件：PCB制版（Altium Design，Allegro Viewer，立创EDA），电路仿真软件（PSPICE），信号仿真软件（HyperLynx）
机器人软件设计：
- 底层控制软件：
	- 技能：C语言，freeRTOS操作系统
	- 软件：单片机编程软件（Keil，VSCode，Ozone等），调试软件（串口调试助手，VOFA，Linux有cutecom，minicom）
- 上层控制软件：需用到ROS2，VScode等
算法设计：
- 自主导航定位技术（传统vs现代）
	- 传统方案：SLAM+路径规划+运动控制
	- 现代方案：深度学习+运动控制
- 感知：主流是机器视觉
	- 图像处理：OpenCV
	- 目标检测/分割算法
		- PyTorch构建神经网络（ski-learn库）
- 定位和建图（SLAM）
	- 框架：ROS2
	- 点云处理：PCL（point cloud library）
	- 其他：Eigen（矩阵运算库）、g2o（图优化库）、ceres（非线性最小二乘库）以及一些处理李群的库
- 运动控制算法
	- LQR，MPC，WBC（优化后的MPC）等算法
	- 全局规划 A* 算法，局部规划 DWA 算法等
	- 传感器融合算法：EKF（扩展卡尔曼滤波），ESKF（状态误差卡尔曼滤波）等

# 控制理论进阶路线
[控制理论——一小时从劝退到入门 - 小方同学](https://zhuanlan.zhihu.com/p/683044170)
## 书籍
入门：
- 主线：Franklin《Feedback Control of Dynamic Systems》
- 参考书：Goodwin《Control System Design》
线性系统：Rugh《Linear System Theory》
非线性系统：
- Khalil《Nonlinear Systems》，标准教科书
- 冯纯伯《非线性控制系统分析与设计》，有实际例子
最优控制：Bryson&何毓琦《Applied Optimal Control》
自适应控制：
- Ioannou《Robust Adaptive Control》，线性自适应系统讲得好
- Krstic《Nonlinear and Adaptive Control Design》，控制领域关门之书
鲁棒控制：
- Dolye 《Feedback control theory》，大师级别
- Skogestad《Multivariable Feedback Control》，最后几章一般
